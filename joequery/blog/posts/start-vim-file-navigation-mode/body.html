{% extends "templates/post.html" %} {% block post %} I often find myself doing the following: 
vim .
to open up VIM and then explore my files. Because I'm lazy, here's a way to open up VIM in file navigation mode if no files are specified when running VIM. Append the following to your .bashrc [bash] function vim(){ vimpath=`which vim` if [ "$#" -gt 0 ] then $vimpath $@ else $vimpath . fi } [/bash] {% endblock post %}
