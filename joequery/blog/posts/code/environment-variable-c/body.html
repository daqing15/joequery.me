If the environment variable name passed in does not exist, `getenv` returns NULL. Using getenv ------------ {{ snippet(p, "getpath.c")|safe }} First, we check the return value of getenv to make sure the environment variable even exists. Then, we use snprintf to make sure our `path` buffer actually has enough space to hold the value of our environment variable. Since snprintf returns the number of bytes that would be written to our `path` buffer if it had infinite space, a simple comparison with `BUFSIZE` lets us know if our buffer can hold the value.
