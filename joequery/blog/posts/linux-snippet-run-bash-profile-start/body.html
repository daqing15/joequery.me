{% extends "templates/post.html" %} {% block post %} If you've edited your ~/.bash_profile to include scripts you'd like executed on start up, you might be surprised when you open up your terminal and those scripts don't execute. By default, ~/.bashrc executes on session start, but ~/.bash_profile doesn't. To execute your ~/.bash_profile on startup, simply append this to your ~/.bashrc 
source ~/.bash_profile

Ta-da!

Now you can print out some cool ascii art on start up, define custom functions (such as mkdir and cd at the same time), and other fun Linux-ey type things. {% endblock post %}
